### 準備するもの

> [brew](https://brew.sh/index_ja)

### インストール

```
$ brew install hugo
```

### hugo サイトを作る

適当なディレクトリで

```
$ hugo new site my_blog
$ cd my_blog
```

### テーマのインストール

> テーマは公式から
> [HUGO](https://themes.gohugo.io/)

### ローカルでテスト

```
$ hugo server -D
```

### 記事の作成

```
$ hugo new posts/markdown.md
```
